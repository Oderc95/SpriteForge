⚙️ Compatibilité

✅ Blender 5.1.1 SpriteForge a été conçu, développé et testé sur Blender 5.1.1. C'est la version officiellement supportée.
⚠️ Autres versions L'utilisation sur d'autres versions de Blender n'est pas garantie. Certaines fonctionnalités peuvent ne pas fonctionner correctement.

Configuration Minimale

  • Blender : 5.1.1+
  • Python : 3.11+
  • RAM : 4 GB minimum (8 GB recommandé)
  • GPU : Optionnel (accélère le rendu)
  • OS : Windows, macOS, Linux

🚀 Installation

1

Télécharger le fichier ZIP

Rendez-vous sur GitHub Releases et téléchargez SpriteForge_v2.1.zip

2

Ouvrir les Préférences

Dans Blender, allez à Edit → Preferences

3

Accéder aux Add-ons

Cliquez sur l'onglet Add-ons dans le panneau des préférences

4

Installer depuis un fichier

Cliquez sur Install from Disk... et sélectionnez le fichier ZIP téléchargé

5

Activer l'addon

Cherchez Cortex SpriteForge et cochez la case pour l'activer

6

Ouvrir le panneau

Allez dans l'onglet Layout, placez votre souris dans la vue 3D, appuyez sur N pour ouvrir la sidebar, puis sélectionnez l'onglet SpriteForge

ℹ️ Installation de PIL/Pillow Si PIL n'est pas installé, SpriteForge l'installera automatiquement au premier lancement. Cela ne prend que quelques secondes.

🧍 Mode Personnages

Utilisez ce mode pour exporter des personnages FBX animés en sprites isométriques multi-directions.

Workflow Personnages

1

Sélectionner le mode

Dans le panneau SpriteForge, choisissez Characters

2

Sélectionner le dossier

Cliquez sur le bouton de dossier et sélectionnez le répertoire contenant vos fichiers FBX, textures et animations

3

Scanner les fichiers

Cliquez sur Scanner pour détecter automatiquement les animations

4

Vérifier la liste

Passez en revue les animations détectées. Vous pouvez retirer celles qui ne vous intéressent pas

5

Importer

Cliquez sur Valider l'import pour importer tous les personnages et animations

6

Configurer l'export

Choisissez la taille du sprite, le format d'export, le nombre de directions, et les paramètres d'éclairage

7

Exporter

Cliquez sur Exporter toutes les actions ou Exporter l'action sélectionnée

Paramètres Clés

Paramètre Description Par Défaut
Taille Sprite Dimension maximale du sprite en pixels (32 à 1024) 128px
Directions Nombre de rotations (1, 4, 8, 16) 8
Format PNG, Spritesheet, GIF Direction, GIF Unique PNG
Frames Plage de frames à exporter Auto-détecté
Fond Transparent Exporte avec alpha channel (transparent) ✅ Activé
💡 Conseil Avant d'exporter, testez le cadrage en basculant à la vue caméra. Vérifiez que votre personnage n'est pas coupé aux bords.

🏠 Mode Assets

Utilisez ce mode pour exporter des modèles statiques (décors, objets, props) en sprites multi-directions.

Formats Supportés

  • FBX - Format standard de l'industrie
  • OBJ - Format de maillage simple
  • GLB/GLTF - Format moderne avec PBR
  • DAE - Collada, format universel

Workflow Assets

1

Sélectionner le mode

Dans le panneau SpriteForge, choisissez Assets

2

Sélectionner le dossier

Sélectionnez le répertoire contenant vos modèles 3D

3

Scanner les modèles

Cliquez sur Scanner pour détecter tous les fichiers 3D

4

Importer les assets

Cliquez sur Valider l'import

5

Configurer les directions

Définissez le nombre de directions (1 pour vue unique, 8 pour 360°)

6

Exporter

Sélectionnez chaque asset et cliquez Exporter, ou exportez tous les assets à la fois

✅ Bon à savoir Le mode Assets utilise un spot lumineux supérieur dédié pour un éclairage optimal des objets. Il s'active automatiquement en mode Assets.

🎨 Mode Pixel Art

SpriteForge intègre un système Pixel Art complet pour créer des sprites rétro ou modernes.

Fonctionnalités Pixel Art

🟪

Grille Pixel

Visualisation en temps réel de la grille pixels sur la caméra. Opacité réglable.

📏

Résolutions

16px à 256px. Choisissez la taille de pixel adaptée à votre jeu.

🎯

Outline

Contour automatique de 1 à 8px. Couleur personnalisable (noir, blanc, etc.).

✨

Dithering

Floyd-Steinberg avec intensité variable. Donne un look rétro authentique.

🔄

Anti-Aliasing

Lissage optionnel des bords. À désactiver pour un rendu pixel parfait.

🎭

Palette

Réduction de palette automatique. Idéal pour les jeux Game Boy ou rétro.

Configurations Recommandées

🎮 Pixel Art Rétro (Zelda, Pokémon)

Paramètre Valeur
Résolution Pixel 32px ou 48px
Outline ✅ Activé (1-2px noir)
Dithering ✅ Activé (60-80%)
Anti-Aliasing ❌ Désactivé

🎨 Pixel Art Moderne (Hyper Light Drifter)

Paramètre Valeur
Résolution Pixel 64px ou 128px
Outline ❌ Désactivé
Dithering ❌ Désactivé
Anti-Aliasing ✅ Activé
💡 Aperçu Activez l'aperçu pixel art pour voir en temps réel le rendu avant l'export. Cliquez sur le bouton "Aperçu pixel art" du panneau.

📷 Caméra Isométrique

SpriteForge crée automatiquement une caméra orthographique optimisée pour les rendus isométriques.

Créer la Caméra

  1. Ouvrez le panneau SpriteForge
  2. Cliquez sur Créer la caméra iso
  3. Un rig caméra est créé automatiquement
  4. La vue s'ajuste à la caméra créée

Contrôles Caméra

Action Bouton Résultat
Zoom +/- Zoom Augmente/diminue la largeur de la caméra
Pan Horizontal Pan ← → Déplace la caméra sur l'axe X
Pan Vertical Pan ↑ ↓ Déplace la caméra sur l'axe Y
Profondeur Z Offset Ajuste la position verticale de la caméra
Vue Caméra Vue Caméra Bascule le viewport à la vue de la caméra

Paramètres Caméra

Largeur (View Width) - Contrôle l'ortho_scale. Augmentez pour voir plus large.
Hauteur (View Height) - Aspect ratio. Ajuste le rapport largeur/hauteur du rendu.
Offset X/Y/Z - Décalage du pivot caméra. Utilisez pour cadrer précisément.
⚠️ Important Ne supprimez pas l'objet "CortexCamRig" après l'avoir créé. C'est le pivot de toute la caméra. Si vous l'avez supprimé, recréez-la avec le bouton "Créer la caméra iso".

⚠️ Erreurs Fréquentes

❌ Le panneau SpriteForge n'apparaît pas

Vérifiez :
  • L'addon est activé dans les préférences
  • Vous êtes dans l'espace de travail Layout
  • Votre souris est dans la vue 3D
  • Vous avez appuyé sur N pour ouvrir la sidebar
  • L'onglet SpriteForge est sélectionné

❌ Les sprites sont coupés aux bords

Solutions :
  • Augmentez la Largeur et Hauteur de la caméra
  • Ajustez les Offsets X/Y pour mieux cadrer
  • Vérifiez que l'objet n'est pas trop grand
  • Testez le cadrage en vue caméra avant d'exporter

❌ Les animations ne sont pas détectées

Assurez-vous que :
  • Le fichier FBX contient une armature
  • Les animations sont définies comme Actions dans Blender
  • Le fichier FBX d'origine a bien les animations en tant que NLA strips
  • Le dossier scanné contient effectivement les fichiers FBX

❌ Le rendu est flou (Pixel Art)

Désactivez l'anti-aliasing :
  • En mode Pixel Art, cochez Pixel Art ✅
  • Décochez Anti-Aliasing ❌
  • Cela donne des pixels nets et parfaits

❌ Les matériaux sont trop brillants

Activez les couleurs mates :
  • Cochez Couleurs mates
  • Cela neutralise les reflets spéculaires et métalliques
  • Idéal pour les assets Meshy ou models générés

❌ PIL/Pillow n'est pas installé

SpriteForge l'installe automatiquement :
  • Attendez le premier lancement (quelques secondes)
  • Vous verrez un message de confirmation
  • Si cela échoue, redémarrez Blender

💡 Conseils Pro

✅ Organisation des dossiers

Organisez vos fichiers par ressource :

Projets/ ├── Personnage_Hero/ │ ├── hero.fbx │ └── animations/ │ ├── walk.fbx │ ├── run.fbx │ └── jump.fbx ├── Objet_Arbre/ │ └── arbre.fbx └── Decor_Maison/ └── maison.obj
✅ Testez le cadrage en premier

Avant d'exporter 100 frames, testez avec 1-2 frames pour vérifier que le cadrage est correct.

✅ Utilisez les sous-dossiers d'export

Cochez "Export aux sous-dossiers" pour que chaque direction soit dans un dossier séparé. Cela facilite l'organisation.

✅ Résolution recommandée

Pour un jeu 2D standard : 64x64px à 128x128px. Pour HD-2D : 256x256px+

✅ Éclairage optimal

Utilisez "Couleurs mates + Lumière principale" pour un rendu cohérent et prêt pour le jeu.

✅ Pixel Art vs. Non-Pixel

Pixel Art = rétro (outline, dithering, grid) • Non-Pixel = moderne (smooth, AA). Choisissez selon votre style.

✅ Animations lissées

Pour les personnages animés, limitez le nombre de frames (30 par défaut) pour éviter un export trop lent.

❓ FAQ

Puis-je exporter des spritesheets ?

Oui. Sélectionnez le format Spritesheet et tous les frames seront organisés dans une grille automatiquement.

Quel est le format de sortie par défaut ?

PNG – Un fichier par frame. Vous pouvez aussi choisir GIF ou Spritesheet.

Combien de directions peuvent être exportées ?

De 1 à 8+ directions. Par défaut : 8 directions (360° ÷ 8 = 45° par direction).

Les Normal Maps sont-elles supportées ?

Oui. Cochez Normal maps pour exporter les cartes de normales (utile pour les moteurs 3D).

Peut-on exporter sans fond transparent ?

Oui. Décochez Fond transparent pour avoir un fond solide (couleur de l'environnement).

Le Pixel Art fonctionne-t-il en temps réel ?

Oui. La grille et les effets pixel art s'affichent en temps réel dans la vue 3D. L'aperçu montre le résultat final.

Quels formats de modèles sont supportés ?

FBX, OBJ, GLB, GLTF, et DAE pour les assets. Pour les personnages : FBX avec armature.

Peut-on exporter plusieurs personnages à la fois ?

Oui. Importez-les tous, puis cliquez Exporter toutes les actions.

Comment optimiser la performance d'export ?

Réduisez la taille du sprite, le nombre de frames, ou le nombre de directions. Désactivez l'anti-aliasing si possible.

Où sont stockés les fichiers exportés ?

Dans le dossier que vous avez défini dans Dossier de sortie. Par défaut : à côté du fichier .blend.

Puis-je personnaliser les couleurs de l'outline ?

Oui. En mode Pixel Art, vous pouvez choisir la couleur de l'outline (noir, blanc, rouge, etc.) avec un color picker.

L'addon fonctionne-t-il sur macOS et Linux ?

Oui. SpriteForge fonctionne sur Windows, macOS et Linux. Blender 5.1.1+ requis.
