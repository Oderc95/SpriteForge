🔥 SpriteForge

L'addon Blender ultime pour créer des sprites 2D isométriques

✓ Blender 5.1.1+ ✓ Python 3.11+ ✓ Multi-plateforme

💡 À propos

SpriteForge transforme automatiquement vos modèles 3D en sprites 2D isométriques prêts pour les jeux vidéo. Importez vos personnages animés ou assets statiques, configurez votre caméra isométrique, et exportez en multi-directions avec un seul clic. Parfait pour les RPG, jeux de stratégie et jeux rétro.

🚀 Installation

1. Télécharger

Récupérez le fichier ZIP depuis les releases GitHub

2. Installer

Édition → Préférences → Add-ons → Installer depuis un fichier

3. Activer

Cherchez « Cortex SpriteForge » et activez l'addon

4. Utiliser

Appuyez sur N dans la vue 3D et cliquez sur SpriteForge

ℹ️ Info : Assurez-vous que Blender 5.1.1+ est installé pour une compatibilité maximale.

🎮 Deux modes d'utilisation

🧍 Mode Personnages

Pour vos FBX animés avec armatures :

  • Détection automatique des animations
  • Synchronisation des frames
  • Export individuel ou global
  • Gestion du root motion
  • Support de multiples personnages

🏠 Mode Assets

Pour vos objets 3D statiques :

  • Cadrage automatique

Dithering

📷 Caméra Isométrique

Caractéristiques

  • Projection orthographique
  • Zoom et pan rapides
  • Décalage XYZ flexible
  • Auto-crop intelligent
  • Prévisualisation temps réel
  • Création automatisée

Utilisation

La caméra est créée automatiquement via le bouton « Créer la caméra ISO ». Utilisez les contrôles pour :

  • Zoomer/dézoomer
  • Déplacer la caméra (pan)
  • Ajuster les offsets X/Y/Z
  • Basculer en vue caméra
  • Prévisualiser le pixel art

⚠️ Dépannage

Le panneau n'apparaît pas

Vérifiez que :

  • L'addon est activé
  • Vous êtes en workspace « Layout »
  • Vous avez appuyé sur « N »
  • L'onglet « SpriteForge » est sélectionné

Les sprites sont coupés

Solutions :

  • Augmenter la largeur/hauteur caméra
  • Ajuster les offsets XYZ
  • Vérifier le modèle en vue caméra

Les animations ne sont pas détectées

Vérifiez que :

  • L'armature est présente
  • Les Actions Blender existent
  • L'export FBX d'origine est valide

Le rendu est flou en Pixel Art

Actions :

  • Désactiver l'anti-aliasing
  • Réduire la résolution pixel
  • Vérifier les paramètres

💡 Conseils professionnels

🎯 Testez le cadrage d'abord – Vérifiez dans la vue caméra avant d'exporter pour éviter les surprises

🎯 Organisez vos dossiers – Un dossier = une ressource (FBX + textures associées)

🎯 Utilisez l'éclairage intégré – C'est plus efficace et rapide que des matériaux complexes

🎯 Exportez en sous-dossiers – Gardez vos rendus organisés par direction et type

🎯 Testez différentes résolutions – 64px, 128px ou 256px selon les besoins de votre jeu
