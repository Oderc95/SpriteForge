Blender 5.1.1

🔥 SpriteForge

Créé par Cortex Game Dev – Artiste, animateur 2D et Pixel Artist.

Important : l'addon a été conçu et testé sur Blender 5.1.1.

Ajoutez votre logo dans assets/logo.png puis remplacez la bannière par votre image.

🎮 Présentation

SpriteForge transforme automatiquement vos modèles 3D en sprites 2D isométriques prêts pour le jeu vidéo. Il permet d'importer des personnages animés ou des assets statiques, de générer des spritesheets, GIFs, PNG individuels et même des rendus Pixel Art.

✨ Fonctionnalités principales

  • Import automatique FBX, OBJ, GLB, GLTF et DAE.
  • Export multi-directions (8 directions par défaut).
  • Spritesheets automatiques.
  • GIF par direction ou GIF global.
  • PNG individuels.
  • Normal Maps exportables.
  • Auto-crop intelligent.
  • Mode Pixel Art avec grille, contour et dithering.
  • Caméra isométrique automatisée.
  • Gestion des sous-dossiers d'origine.
  • Éclairage optimisé pour personnages et assets.

⚙️ Installation

  1. Télécharger le fichier ZIP de l'addon.
  2. Dans Blender : Édition → Préférences → Add-ons.
  3. Cliquer sur Installer depuis un fichier.
  4. Sélectionner le ZIP.
  5. Activer Cortex SpriteForge.
  6. Ouvrir le panneau : Vue 3D → Sidebar → SpriteForge.

🚀 Workflow recommandé

  1. Créer la caméra SpriteForge.
  2. Importer vos personnages ou assets.
  3. Vérifier le cadrage.
  4. Choisir la taille du sprite.
  5. Configurer les directions.
  6. Activer Pixel Art si nécessaire.
  7. Lancer l'export.

🧍 Mode Personnages

Le mode Personnages est conçu pour les FBX animés.

  • Scan automatique du dossier.
  • Détection des animations.
  • Synchronisation automatique des frames.
  • Export individuel ou global.
  • Gestion du root motion.

🏠 Mode Assets

  • Import d'objets statiques.
  • Rotation automatique sur plusieurs directions.
  • Optimisé pour décors, objets et props.
  • Éclairage dédié avec Spot supérieur.

🟪 Pixel Art

  • Grille de visualisation temps réel.
  • Résolutions : 16 à 256 px.
  • Contour automatique.
  • Dithering Floyd-Steinberg.
  • Réduction NEAREST pour des pixels nets.
  • Anti-aliasing optionnel.
💡 Pour un rendu rétro authentique, désactivez l'anti-aliasing et activez un contour noir de 1 px.

📷 Caméra Isométrique

SpriteForge crée automatiquement une caméra orthographique optimisée pour les sprites isométriques.

  • Zoom rapide.
  • Pan caméra.
  • Décalage X/Y/Z.
  • Recadrage automatique.
  • Prévisualisation Pixel Art.

📦 Formats d'export

  • PNG : une image par frame.
  • Spritesheet : toutes les animations regroupées.
  • GIF Direction : un GIF par angle.
  • GIF Unique : toutes les directions dans une seule animation.
  • Normal Maps : génération de cartes normales.

⚠️ Erreurs à éviter

  • Ne pas utiliser une version Blender inférieure à 5.1.1.
  • Éviter les animations sans armature valide.
  • Ne pas supprimer la caméra SpriteForge après configuration.
  • Vérifier les chemins d'export avant le rendu.
  • Ne pas mélanger plusieurs personnages dans le même export sans contrôle du cadrage.
  • Attention aux modèles avec matériaux métalliques excessifs.
